「译」CSS 3D 注意事项

  2016-10-01     前端

原文链接：Things to Watch Out for When Working with CSS 3D 作者 ANA TUDOR

我一直挺喜欢 3D 几何。在注意到 CSS 支持得越来越全面的同时，我也开始使用 CSS 3D 变换（transform）。然而在创建多面体时，我自然地在 3D 中使用 transform 去创建 2D 图形并移动、旋转它们，一开始有些东西就绊倒了我。我想我该记录一下我遇见过的奇特的东西，希望你能绕过这些障碍。
