    • 网络配置与服务这门课对我来说已经成为周二晚的噩梦了。
    在下周二之前要做的事情有点多：

    1. dns server 正向反向的解析
    2. apache 服务器
      1. 重设置文档主目录
      2. 只监听80和81端口
      3. 主目录默认打开 php jsp html文档
      4. 设置默认字符集为 GB2312
      5. 建立虚拟目录 OA: /var/www/OA
      6. 访问目录时需要用户认证，只认证 teacher
