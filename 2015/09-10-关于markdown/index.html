Mou支持的快捷键

粗体 － Bold － cmd ＋ B
斜体 － Emphasize － cmd ＋ I
行间代码区 － Inline Code － cmd ＋ K
链接 － Link － ctl ＋ shift ＋ L
图片 － Picture － ctl ＋ shift ＋ I
提行 － New Line － shift ＋ return
评注 － Comment － cmd ＋ ／

这总不算烂尾了吧？
